Database

Tidak memerlukan MongoDb ataupun web database lain karena Panel mampu menyimpan semua database project bot mu.

Secure

Keamanan di prioritaskan guna mendapatkan kepercayaan user.

Configuration

Versi Node.js bisa di ubah dan dikonfigurasi kan sesuai keinginan di startup dockerimage. Ada beberapa pilihan versi nodejs antara lain 16, 17, 18, 19

Owner Setting

Biasanya setiap bot mempunyai settingan Owner masing masing yang di namai settings.js, config.js, config.json etc.js dan lain lain.
